# Global Restaurants Data Analysis

This project provides an analysis of a global restaurants dataset to identify top-rated restaurants, the most expensive restaurants, cities with the most restaurants, and the most common cuisines. The analysis is performed using **Python**, **pandas**, and **matplotlib**.

- `Cuisines`: Types of cuisines offered  
- `Average_Cost_for_two`: Average cost for two people
